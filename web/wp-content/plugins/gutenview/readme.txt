Make the WordPress block editor's hidden editing affordances visible and discoverable.

== Description ==

GutenView is a lightweight enhancement for the WordPress block editor. It makes
editing affordances that are normally hidden behind hover states and precise
mouse positioning clearly visible, so non-technical editors can find how to add,
select, and edit blocks without hunting.

It is not a page builder. Gutenberg stays Gutenberg; GutenView simply removes
friction around it. Disable the plugin and the editor works exactly as before.

This is an early scaffold. Features are being added incrementally.

== Installation ==

1. Upload the `gutenview` folder to `/wp-content/plugins/`.
2. Activate the plugin through the "Plugins" screen in WordPress.
